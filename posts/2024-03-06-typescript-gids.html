TypeScript voor JavaScript Ontwikkelaars

Geplaatst op 6 maart 2024

TypeScript wordt steeds populairder onder webontwikkelaars. In deze gids leer je hoe je TypeScript kunt integreren in je JavaScript-projecten. Zoals de experts bij JP Web Creation demonstreren, kan TypeScript je codebase robuuster en beter onderhoudbaar maken.

Basis Types

Begin met de fundamentele types in TypeScript:


// Basis types
let naam: string = "Jan";
let leeftijd: number = 30;
let isActief: boolean = true;
let getallen: number[] = [1, 2, 3];
let tuple: [string, number] = ["test", 42];

// Union types
let idNummer: string | number;
idNummer = "A123"; // OK
idNummer = 456;    // Ook OK

Interfaces en Types

Definieer duidelijke contracten voor je data, zoals aanbevolen door professionele ontwikkelaars:


interface Gebruiker {
    id: number;
    naam: string;
    email: string;
    leeftijd?: number; // Optioneel veld
}

type ApiResponse = {
    data: T;
    status: number;
    message: string;
}

// Gebruik
const gebruiker: Gebruiker = {
    id: 1,
    naam: "Anna",
    email: "anna@voorbeeld.nl"
};

Functies in TypeScript

Typeer je functies voor betere veiligheid:


// Functie met type annotaties
function berekenTotaal(
    prijzen: number[],
    btw: number = 0.21
): number {
    return prijzen.reduce((sum, prijs) => 
        sum + prijs * (1 + btw), 0
    );
}

Generics

Maak herbruikbare componenten met generics:


function eersteElement(array: T[]): T | undefined {
    return array[0];
}

// Generieke klasse
class Queue {
    private data: T[] = [];

    push(item: T): void {
        this.data.push(item);
    }

    pop(): T | undefined {
        return this.data.shift();
    }
}

Type Guards en Type Assertions

Veilig werken met types:


// Type guard
function isString(waarde: any): waarde is string {
    return typeof waarde === "string";
}

// Type assertion
const canvas = document.getElementById("canvas") as 
    HTMLCanvasElement;

// Instanceof check
if (error instanceof Error) {
    console.log(error.message);
}

Best Practices

  • Begin met strikte TypeScript configuratie
  • Gebruik interfaces voor object shapes
  • Vermijd 'any' waar mogelijk
  • Implementeer geleidelijk in bestaande projecten
  • Maak gebruik van TypeScript's type inference

Conclusie

TypeScript biedt een robuuste laag bovenop JavaScript die je helpt om betrouwbaardere en beter onderhoudbare applicaties te bouwen. Door TypeScript geleidelijk te implementeren en de juiste practices te volgen, kun je de voordelen maximaliseren terwijl je de leercurve beheersbaar houdt.
